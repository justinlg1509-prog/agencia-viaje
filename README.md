# SATW - Spain Around The World 🌍🇪🇸

**SATW** es una plataforma web moderna y atractiva diseñada para promocionar el turismo en España. El sitio conecta a viajeros de todo el mundo con los destinos más emblemáticos y servicios exclusivos del país, ofreciendo una experiencia visualmente inmersiva y totalmente adaptativa (responsive).

## ✨ Características

* **Diseño Moderno:** Interfaz limpia con efectos de desenfoque (*backdrop-filter*), degradados dinámicos y animaciones suaves.
* **Totalmente Responsive:** Optimizado para dispositivos móviles, tablets y ordenadores de escritorio.
* **Menú Navegación Inteligente:** Menú "hamburger" funcional en móviles implementado exclusivamente con CSS (sin necesidad de JavaScript).
* **Secciones Informativas:**
    * **Hero Section:** Impactante bienvenida con imagen de fondo y llamada a la acción.
    * **Destinos:** Cuadrícula de tarjetas con efectos de escala que muestran lugares destacados como Barcelona, Madrid y Sevilla.
    * **Servicios:** Iconografía clara detallando el valor añadido de la agencia.
    * **Testimonios:** Espacio para la prueba social y opiniones de clientes.
    * **Formulario de Contacto:** Diseño intuitivo para la captación de leads.

## 🛠️ Tecnologías utilizadas

* **HTML5:** Estructura semántica para SEO y accesibilidad.
* **CSS3:** * **Flexbox & CSS Grid:** Para una disposición de elementos flexible.
    * **Media Queries:** Para garantizar la adaptabilidad en cualquier pantalla.
    * **Animaciones Keyframes:** Para la entrada suave de contenidos.
    * **Variables y Filtros:** Uso de filtros de desenfoque y capas de color.
